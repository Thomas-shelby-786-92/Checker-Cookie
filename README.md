# Cookie Checker
Free Open Source Cookie Checker For Everyone

# Availability
- Mac 
- Linux

# Installation
Run Following Commands
`sudo apt update`
`sudo apt install python3`
`pip install -r requirements.txt`

# Usage
- Place your Cookie files inside `cookie` folder
- Run `python3 checker.py`
